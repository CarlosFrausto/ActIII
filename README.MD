# Sistema de Inventario para Tienda

## Descripción

Sistema desarrollado en Python que permite administrar un catálogo de productos, controlar el inventario y gestionar ventas.

La carpeta tienda tiene el código, la carpeta diagramas tiene los PUML.

## Funciones

- Registrar productos
- Actualizar stock
- Procesar ventas
- Cancelar ventas
- Mostrar productos agotados
- Mostrar producto más vendido
- Ordenar por precio
- Ordenar por stock
- Ordenar por ventas

## Requisitos

Python 3

## Ejecución

```bash
python main.py
```

## Estructura

- producto.py
- inventario.py
- main.py
- diagramas/
